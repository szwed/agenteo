# Apeiro Reference Architecture

ApeiroRA provides concepts, specs, and building blocks for a **Platform Mesh** -- a consistent environment for managing and accessing diverse services from different providers. Supports infrastructure (VMs), runtimes (K8s clusters), application services (databases), and business services.

Developers acquire services through the Platform Mesh. Service suppliers use it as a marketplace across consumers. Works for public, private, and edge cloud at any scale.

# Data Fabric

Architectural approach for flexible, reusable data access across an organization (per Gartner definition).

## Key Elements

- **Design Concept, Not a Product** - architectural approach involving various technologies
- **Integrated Layer** - connects data sources, applications, and users across platforms/locations
- **Metadata-Driven** - active metadata (real-time, contextual, enriched) automates data management
- **Continuous Analytics** - constantly analyzes metadata and usage patterns to optimize data delivery
- **Environment-Agnostic** - works across on-premises, cloud, hybrid, multi-cloud

## Apeiro Focus

Within Apeiro, Data Fabric focus is on **metadata**. Decentralized self-describing of application resources enables identification of correct business objects behind exposed resources, clarifies meaning, and determines which integration points link to the same underlying object.
